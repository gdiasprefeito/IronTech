# IronTech
Site de armaduras do Tony Stark 
1. Página Inicial

A página inicial será chamativa e interativa, apresentando:

Introdução sobre Din Djarin
Destaque dos principais trabalhos realizados
Exibição de seus equipamentos mandalorianos
Apresentação de sua nave espacial Razor Crest
Layout visual inspirado no universo Star Wars


2. Portfólio — Melhores Trabalhos

Página dedicada aos melhores serviços e missões concluídas por Din Djarin.

Conteúdo:

Lista de caçadas e missões importantes
Imagens ilustrativas
Pequena descrição de cada trabalho realizado
Destaque para contratos de maior dificuldade


3. Galeria do Grogu

Página com fotos e imagens do Grogu.

Conteúdo:

Galeria de imagens
Layout organizado em grade (opcional)
Efeitos visuais ao passar o mouse nas imagens(opcional)


4. História de Din Djarin e Grogu

Página contando a relação entre Din Djarin e Grogu.

Conteúdo:

Como eles se conheceram
Desenvolvimento da amizade entre os dois
Texto informativo com imagens ilustrativas


5. Formulário de Solicitação de Serviços

Página para contratação dos serviços de Din Djarin.

Campos do formulário:

Nome
Tipo de missão
Descrição do serviço
Botão de envio

Objetivo:
Permitir que visitantes solicitem serviços de caça de recompensas.
